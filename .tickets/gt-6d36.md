# Build agent-grade CLI reference and instructions surface

Create a first-class agent-facing CLI reference for go-ticket so agents can learn supported commands, safe workflows, argument order, examples, machine-readable surfaces, and common recovery patterns without scraping terse help text or guessing from failures.

## Acceptance Criteria

A command or documented surface such as gtk reference, gtk instructions, or equivalent provides agent-grade command coverage; output works without an active .tickets directory where practical; examples cover create/show/list/query/ready/blocked/dependencies/notes/closeout; risky or unsupported behaviors are named clearly; tests cover command-specific help/reference output; README/docs point agents and editor integrations at the reference; post-implementation security review confirms the reference does not encourage unsafe shell, path, or config patterns.
